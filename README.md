# Java-Linear-Regression
Fish.csv ile basit lineer regresyon
## Projenin Özellikleri
* Veri Okuma: 'Fish.csv' dosyasından verileri dinamik olarak okur
* Shuffle: Veri setini rastgele karıştırarak tarafsız bir eğitim sağlar
* Eğitim ve Test Ayrımı: Verinin %70' ini eğitim, %30' unu test seti olarak ayırır.
* Korelasyon Analizi: 5 farklı bağımsız değişkenin bağımlı değişken (Y) ile korelasyonunu hesaplar.
* Otomatik Model Seçimi: En yüksek korelasyona sahip değişkeni otomatik olarak seçer ve model kurulur.
* Performans Ölçümü: Eğitim ve Test setleri için SSE(Sum of Squared Errors) değerlerini hesaplar.
